# Login to the app

After starting the application, the login screen will appear, enter your username and password in the appropriate fields and press the `Log in` button.

If you forget your password, select the `Forgot password` button and enter your username on the next screen. An email with a link to create a new password will be sent to the email address linked with the user.

At the top right of the screen there is a settings icon where you can change the language. Currently available languages are English and Polish. The application will launch with the device's system language by default.
